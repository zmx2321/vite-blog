1. и®ӨиҜҶLinux вҖӢ

linuxжҳҜд»Җд№Ҳ вҖӢ

  • linuxжҳҜдёҖдёӘж“ҚдҪңзі»з»ҹпјҢе®ғдёҚжҳҜдёҖдёӘжҲҗеһӢзҡ„дёңиҘҝпјҢе®ғзҡ„жң¬иҙЁжҳҜеҶ…ж ёвҖңkernelвҖқгҖӮ
    • kernelдё»иҰҒжҳҜз”ЁcеҶҷзҡ„пјҢйҮҢйқўжңүеҗ„з§Қз®—жі•е’ҢеӨ„зҗҶжөҒзЁӢгҖӮ
    • еҶ…ж ёеҜ№жҖ§иғҪиҰҒжұӮйқһеёёй«ҳпјҢеӣ дёәеә•еұӮзҡ„дёңиҘҝйңҖиҰҒеҗ‘дёҠжҸҗдҫӣеҗ„з§ҚapiпјҢжҲ–и®ёдёҖдёӘapiжҜҸз§’й’ҹдјҡжү§иЎҢжҲҗеҚғдёҠдёҮж¬ЎпјҢж“ҚдҪңйқһеёёйў‘з№ҒпјҢйңҖиҰҒеҫҲй«ҳзҡ„ж•ҲзҺҮгҖӮ
  • жҲ‘д»¬е№іж—¶жүҖиҜҙзҡ„UbuntuгҖҒCentOSйӮЈдәӣжҳҜжҢҮеҸ‘иЎҢзүҲпјҢlinuxеҶ…ж ёеҸӘжҳҜжӢҘжңүжңҖеҹәжң¬зҡ„еҠҹиғҪпјҢе®ғжң¬иә«жңүе…ӯеӨ§зі»з»ҹпјҢиҝҷе…ӯеӨ§зі»з»ҹе°ұз»„жҲҗдәҶж“ҚдҪңзі»з»ҹж ёеҝғзҡ„еҠҹиғҪгҖӮ
  • дҪҶжҳҜйӮЈдәӣеҸ‘иЎҢзүҲпјҢйҷӨдәҶиҝҷе…ӯеӨ§зі»з»ҹжүҖжӢҘжңүзҡ„еҠҹиғҪд»ҘеӨ–пјҢиҝҳеҠ дёҠдәҶеҫҲеӨҡеӨ–еЈіжҖ§зҡ„дёңиҘҝпјҢиҝҳжңүеҗ„з§ҚиҪҜд»¶пјҢеҗ„з§ҚжңҚеҠЎзӯүзӯүпјҢиҝҷж ·жүҚжһ„жҲҗдәҶе®Ңж•ҙзҡ„linuxгҖӮ

UNIXгҖҒlinuxдёҺwindowsзҡ„еҢәеҲ« вҖӢ

  • UNIXж“ҚдҪңзі»з»ҹжҳҜдёҖдёӘзӣёеҪ“еҸӨиҖҒзҡ„зі»з»ҹпјҢе®ғеҘ е®ҡдәҶзҺ°д»Јж“ҚдҪңзі»з»ҹзҡ„еҹәзЎҖпјҢе®ғеҗҢж—¶д№ҹеҲ¶е®ҡдәҶеҫҲеӨҡдёҡз•Ңзҡ„и§„иҢғгҖӮ
    • macжҳҜunixеҶ…ж ёпјҢжҳҜBSDеҶ…ж ёпјҢlinuxжҳҜеҸҰеӨ–дёҖз§ҚеҶ…ж ёпјҢBSDеҶ…ж ёжҳҜжңүзүҲжқғзҡ„гҖӮ
    • unixе®ғзҡ„lience(и®ёеҸҜиҜҒ)е’Ңlinuxе®ҡд№үзҡ„жҳҜдёҚдёҖж ·зҡ„пјҢејҖжәҗиҪҜд»¶зҡ„lienceеҗ„жңүзү№зӮ№пјҢзәҜзІ№зҡ„unixзҡ„зүҲжқғжҳҜеҪ’еұһдәҺиҙқе°”е®һйӘҢе®Өзҡ„пјҢиҙқе°”е®һйӘҢе®ӨжҳҜAT&Tж——дёӢзҡ„пјҲзҫҺеӣҪзҡ„дёҖдёӘеӨ§йҖҡи®Ҝе…¬еҸёпјүгҖӮ
  • windowsдёҖиҲ¬жҳҜз”ЁеңЁе®ўжҲ·з«Ҝзҡ„жЎҢйқўдёҠпјҢдҪҶжҳҜд№ҹеҒ¶е°”иў«з”ЁеҒҡжңҚеҠЎеҷЁпјҢз”ЁеҒҡжңҚеҠЎеҷЁзҡ„windowsд№ҹжҳҜжңүеӣҫеҪўз•Ңйқўзҡ„пјҢдҪҶе…¶е®һжңүеӣҫеҪўз•Ңйқўзҡ„жңҚеҠЎеҷЁжҳҜеҜ№зі»з»ҹиө„жәҗзҡ„жөӘиҙ№гҖӮ
  • linuxдёҖиҲ¬жҳҜз”ЁеңЁжңҚеҠЎеҷЁдёҠпјҢеҹәжң¬дёҚдҪҝз”ЁеӣҫеҪўз•ҢйқўпјҢеӣ дёәжңҚеҠЎеҷЁе®ғеҜ№жҖ§иғҪиҰҒжұӮеҫҲй«ҳгҖӮlinuxзҡ„еҶ…ж ёжҳҜд»ҺunixдёҠз»§жүҝдёӢжқҘзҡ„пјҢе®ғдё»еј д»Ҙз®ҖдёәзҫҺгҖӮwindowsеҲҷжҜ”иҫғеӨҚжқӮгҖӮ

еёёз”Ёзҡ„linuxеҸ‘иЎҢзүҲ вҖӢ

  • еёёз”ЁпјҡUbuntu(дёҖе№ҙеҸ‘иЎҢ2ж¬Ў[4гҖҒ10]lts)гҖҒCentOS(зәўеёҪзҡ„зӨҫеҢәз»ҙжҠӨзүҲ)гҖҒFedora
  • ж¬§зҫҺеёёз”ЁпјҡDeblan
  • дёәе®үе…Ёж–№йқўејҖеҸ‘зҡ„пјҡKali(жңүеҫҲеӨҡзҪ‘з»ңе®үе…Ёе·Ҙе…·е’Ңй»‘е®ўе·Ҙе…·)
  • жңҚеҠЎеҷЁеёӮеңәеҚ жңүжҜ”дҫӢиҫғеӨ§пјҡCentOSгҖҒRed Hat(зәўеёҪ[е•Ҷдёҡ])
  • жЎҢйқўзі»з»ҹеёӮеңәеҚ жңүжҜ”дҫӢиҫғеӨ§пјҡUbuntuгҖҒDeblanгҖҒFedora

2. е®үиЈ…LinuxзҺҜеўғ вҖӢ

жіЁж„ҸдәӢйЎ№ вҖӢ

  1. жҺЁиҚҗе®үиЈ…CentOSпјҢжңүDVDе’ҢMinimal(иҝ·дҪ зүҲ)пјҢжҺЁиҚҗе®үиЈ…иҝ·дҪ зүҲгҖӮ
  2. е®үиЈ…CentOSйңҖиҰҒдҫқиө–иҷҡжӢҹжңәпјҢжҺЁиҚҗдҪҝз”ЁVMwareпјҢеңЁдёӢиҪҪдёӢйқўйҖүжӢ©вҖңWorkstation ProвҖқпјҢз„¶еҗҺжүҫдёҖдёӘжіЁеҶҢз ҒжіЁеҶҢе°ұеҸҜд»ҘдәҶпјҢдҪҶжҳҜжіЁж„ҸдёҚиҰҒеҚҮзә§пјҢеҗҰеҲҷдјҡеӨұж•ҲгҖӮ
  3. иҷҡжӢҹжңәзҺҜеўғпјҡ
    В В  a. еҶ…еӯҳпјҡ еҰӮжһңеёҰеӣҫеҪўз•ҢйқўпјҢжҺЁиҚҗеҶ…еӯҳиҮіе°‘2gпјҢеҰӮжһңжҳҜиҝ·дҪ зүҲзҡ„CentOSпјҢеӣ дёәд»–еҸӘжңү500MпјҢжҺЁиҚҗеҶ…еӯҳиҮіе°‘1gгҖӮ
    В В  b. зҪ‘з»ңйҖӮй…ҚеҷЁпјҡ й…ҚзҪ®зҪ‘з»ңзҡ„ж—¶еҖҷдёҚиҰҒдҪҝз”ЁNATпјҢеҗҰеҲҷзҪ‘з»ңзҺҜеўғе°ұдјҡйқһеёёеӨҚжқӮдәҶпјҢе®ғдјҡз»ҷдҪ иҮӘеҠЁй…ҚзҪ®дёҖдёӘиҷҡжӢҹзҡ„еұҖеҹҹзҪ‘жқҘпјҢеӨ„зҗҶдәӢжғ…зҡ„ж—¶еҖҷдјҡдә§з”ҹдёҖдәӣе№Іжү°гҖӮжҺЁиҚҗдҪҝз”ЁжЎҘжҺҘжЁЎејҸгҖӮдҪҶдҪҝз”ЁжЎҘжҺҘжЁЎејҸпјҢеҝ…йЎ»иҰҒдҝқиҜҒзҪ‘з»ңзҡ„з•…йҖҡпјҢдёҖж—ҰжҠҠзҪ‘з»ңж–ӯејҖпјҢжЎҘжҺҘе°ұдјҡеӨұж•Ҳ
    В В  c. дё“й—Ёзҡ„зҪ‘з»ңз®ЎзҗҶпјҡ дё“й—Ёзҡ„зҪ‘з»ңз®ЎзҗҶеңЁзј–иҫ‘дёӯзҡ„иҷҡжӢҹзҪ‘з»ңзј–иҫ‘еҷЁдёӯпјҢе®ғй»ҳи®Өй…ҚдәҶдёӨдёӘиҷҡжӢҹзҪ‘з»ңйҷӨйқһжҳҜеёҢжңӣй…ҚзҪ®дёҖдёӘе’ҢеӨ–йғЁйҡ”з»қзҡ„дёҖдёӘиҷҡжӢҹзҪ‘
  4. дёӢиҪҪcentosй•ңеғҸжҺЁиҚҗпјҡ
    дёҖиҲ¬еҰӮжһңдҪҝз”ЁеӣҪеӨ–зҡ„й“ҫжҺҘең°еқҖдёӢиҪҪй•ңеғҸдјҡжҜ”иҫғж…ўпјҢиҝҷйҮҢжҺЁиҚҗдҪҝз”ЁеӣҪеҶ…зҡ„дёӢиҪҪй•ңеғҸзҡ„ең°еқҖпјҡ163гҖҒjdcloudгҖҒaliyunгҖҒhuaweicloud
  5. жҜ”иҫғиҖҒзҡ„жңәеӯҗеҸҜиғҪдёҚж”ҜжҢҒиҷҡжӢҹеҢ–пјҡ
    иҝӣе…ҘBIOS(дё»жқҝ)пјҢиҝӣе…ҘSystem ConfigurationпјҢжүҫеҲ°Virtualization TechnologyпјҢйҖүжӢ©Enabled

  1. зЎ¬д»¶е…је®№жҖ§ -> йҖүжӢ©жңҖж–°зүҲ
  2. е®үиЈ…зЁӢеәҸе…үзӣҳжҳ еғҸж–Үд»¶ -> йҖүжӢ©й•ңеғҸж–Үд»¶пјҲеӣәе®ҡи·Ҝеҫ„пјү
    • зӨәдҫӢпјҡвҖңD:\vm\systemOS\centos-7\xxxxвҖқ
  3. иҷҡжӢҹжңәеҗҚз§° -> иҷҡжӢҹжңәеңЁе·Ұиҫ№жҳҫзӨәзҡ„еҗҚз§°
  4. дҪҚзҪ® -> иҝҷжҳҜдёҖдёӘеӨ§еқ‘пјҢиҝҷйҮҢзҡ„дҪҚзҪ®жҳҜжҢҮпјҢеңЁе®үиЈ…еҘҪдәҶиҷҡжӢҹжңәд№ӢеҗҺпјҢиҰҒеңЁзЎ¬зӣҳдёҠз”ҹжҲҗдёҖдёӘйқһеёёеӨ§зҡ„ж–Үд»¶пјҢе®һйҷ…дёҠиҝҷдёӘж–Үд»¶жҳҜдёҖдёӘиҷҡжӢҹзЈҒзӣҳпјҢиҮіе°‘иҰҒжңү10gзҡ„еҜҢдҪҷз©әй—ҙгҖӮдёҖиҲ¬дёҚеҚ з”Ёзі»з»ҹз©әй—ҙпјҲCзӣҳпјүгҖӮ
    • зӨәдҫӢпјҡвҖңD:\vm\sys_space\CentOS7\vmspace\xxxвҖқ
  5. еӨ„зҗҶеҷЁж•°йҮҸ -> 1
  6. жҜҸдёӘеӨ„зҗҶеҷЁзҡ„еҶ…ж ёж•°йҮҸ -> 1-2
    • еӨ„зҗҶеҷЁе°ұжҳҜcpuпјҢжҲ‘зҡ„и®Ўз®—жңәеҸӘжңүдёҖдёӘеӨ„зҗҶеҷЁ(cpu)пјҢ6дёӘеҶ…ж ёгҖӮ
    • иҝҷйҮҢзҡ„еӨ„зҗҶеҷЁж•°йҮҸе’ҢеҶ…ж ёж•°йҮҸе№¶дёҚжҳҜи¶ҠеӨҡи¶ҠеҘҪпјҢдёҖиҲ¬иҝҷйҮҢзҡ„йҖүжӢ©иҰҒе°ҸдәҺдҪ зҡ„и®Ўз®—жңәжң¬иә«зҡ„cpuе’ҢеҶ…ж ёпјҢе®ғеҸ–еҶідәҺдҪ зҡ„и®Ўз®—жңәгҖӮ
  7. жӯӨиҷҡжӢҹжңәзҡ„еҶ…еӯҳ -> 2048MB

  9. дёӢйқўзҡ„жҺ§еҲ¶еҷЁгҖҒиҷҡжӢҹзЈҒзӣҳйӮЈдәӣйғҪйҖүй»ҳи®Ө

  12. йҖүжӢ©е°ҶиҷҡжӢҹзЈҒзӣҳеӯҳеӮЁдёәеҚ•дёӘж–Үд»¶
    • еҲҶжӢҶзҡ„ж„Ҹд№үеңЁдәҺжҠҠиҷҡжӢҹзЈҒзӣҳеҲҶжҲҗиӢҘе№ІдёӘпјҢиҝҷж ·жӢ·иҙқзҡ„ж—¶еҖҷйҖҹеәҰдјҡеҝ«дёҖдәӣпјҢиҝҷжҳҜз”ұеә•еұӮзҡ„IOи®ҫеӨҮзҡ„зү№зӮ№жқҘеҶіе®ҡзҡ„пјҢдҪҶдёҖиҲ¬еҲӣе»әдәҶиҷҡжӢҹжңәд№ӢеҗҺпјҢдёҚдјҡеҺ»еҠЁд»–пјҢеӯҳеӮЁдёәеҚ•дёӘж–Үд»¶пјҢжҖ§иғҪеҸҚиҖҢдјҡй«ҳдёҖдәӣгҖӮ
  13. жңҖеҗҺе®ғе°ұејҖе§ӢеҲӣе»әзЈҒзӣҳж–Үд»¶дәҶгҖӮ

4. еүҚз«ҜејҖеҸ‘еҝ…йЎ»иҰҒжҮӮзҡ„зҹҘиҜҶ вҖӢ

зҪ‘з»ңз«ҜеҸЈ вҖӢ

  • д»Җд№ҲжҳҜз«ҜеҸЈ
    • еҰӮжһңжғійҖҡиҝҮзҪ‘з»ңеҺ»и®ҝй—®еҸҰеӨ–дёҖеҸ°з”өи„‘пјҢеҝ…йЎ»еҫ—еҺ»ејҖдёҖдёӘй—ЁпјҢиҝҷдёӘй—ЁпјҢе®һйҷ…дёҠж“ҚдҪңзі»з»ҹдёҠжңүеҫҲеӨҡй—ЁпјҢе®ғд»¬йғҪжңүе®ғиҮӘе·ұзҡ„з”ЁйҖ”пјҢжҲ‘д»¬з©¶з«ҹйңҖиҰҒејҖд»Җд№Ҳй—ЁпјҢйңҖиҰҒеңЁдёҠйқўзј–дёҖдёӘеҸ·пјҢhttpиҝҷдёӘй—ЁжҳҜ80еҸ·й—ЁзӯүзӯүпјҢиҝҷдәӣй—ЁдёҚиғҪд№ұзӘңпјҢеҰӮжһңд№ұзӘңпјҢе°ұдјҡеҸ‘з”ҹз«ҜеҸЈеҶІзӘҒгҖӮ
    • жҚўеҸҘиҜқиҜҙпјҢдёҖдёӘз«ҜеҸЈпјҢеңЁеҗҢдёҖдёӘж—¶й—ҙеҶ…пјҢеҸӘиғҪиў«дёҖдёӘиҝӣзЁӢеҺ»дҪҝз”ЁгҖӮ
  • й»ҳи®Өз«ҜеҸЈ
    • httpй»ҳи®Өз«ҜеҸЈ80
    • httpsй»ҳи®Өз«ҜеҸЈ 443
    • http д»ЈзҗҶз«ҜеҸЈ 8080

д»Җд№ҲжҳҜжңҚеҠЎ вҖӢ

  • еңЁlinuxдёӢжҹҘзңӢжңҚеҠЎ
    • systemctl => жҢүctrl+cйҖҖеҮә
    • systemctl status nginx => жҹҘзңӢnginxжңҚеҠЎзҠ¶жҖҒ
    • systemctl start nginx => ејҖеҗҜжңҚеҠЎ
    • systemctl stop nginx => еҒңжӯўжңҚеҠЎ
    • systemctl restart nginx => йҮҚеҗҜжңҚеҠЎ
    • nginx -s reload => дёҚеҒңжӯўжңҚеҠЎпјҢйҮҚж–°иҜ»еҸ–дёҖж¬Ў

д»Җд№ҲжҳҜз»Ҳз«Ҝ вҖӢ

  • з»Ҳз«ҜпјҲTerminalпјүд№ҹз§°з»Ҳз«Ҝи®ҫеӨҮпјҢжҳҜи®Ўз®—жңәзҪ‘з»ңдёӯеӨ„дәҺзҪ‘з»ңжңҖеӨ–еӣҙзҡ„и®ҫеӨҮпјҢдё»иҰҒз”ЁдәҺз”ЁжҲ·дҝЎжҒҜзҡ„иҫ“е…Ҙд»ҘеҸҠеӨ„зҗҶз»“жһңзҡ„иҫ“еҮәзӯүгҖӮ
  • еңЁж—©жңҹи®Ўз®—жңәзі»з»ҹдёӯпјҢз”ұдәҺи®Ўз®—жңәдё»жңәжҳӮиҙөпјҢеӣ жӯӨдёҖдёӘдё»жңәпјҲIBMеӨ§еһӢи®Ўз®—жңәпјүдёҖиҲ¬дјҡй…ҚзҪ®еӨҡдёӘз»Ҳз«ҜпјҢиҝҷдәӣз»Ҳз«Ҝжң¬иә«дёҚе…·еӨҮи®Ўз®—иғҪеҠӣпјҢд»…д»…жүҝжӢ…дҝЎжҒҜиҫ“е…Ҙиҫ“еҮәзҡ„е·ҘдҪңпјҢиҝҗз®—е’ҢеӨ„зҗҶеқҮз”ұдё»жңәжқҘе®ҢжҲҗгҖӮ
  • еңЁдёӘдәәи®Ўз®—жңәж—¶д»ЈпјҢдёӘдәәи®Ўз®—жңәеҸҜд»ҘиҝҗиЎҢз§°дёәз»Ҳз«Ҝд»ҝзңҹеҷЁзҡ„зЁӢеәҸжқҘжЁЎд»ҝдёҖдёӘз»Ҳз«Ҝзҡ„е·ҘдҪңгҖӮ

  • з„¶еҗҺиҫ“е…ҘеҜҶз ҒпјҢиҫ“е…ҘеҜҶз Ғзҡ„ж—¶еҖҷе®ғдёҚеӣһжҳҫпјҢеӣ дёәе®үе…ЁжҖ§й—®йўҳпјҢеҲ«дәәдёҚзҹҘйҒ“дҪ зҡ„еҜҶз ҒдјҡжңүеӨҡе°‘дҪҚгҖӮ

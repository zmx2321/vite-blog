# chrome相关笔记

## Chrome书签手动同步方法
- 保持登陆状态
- 在地址栏输入 `chrome://sync`
- 点击 Stop Sync (Keep Data)
- 点击 Request Start
- 注意：在同步书签时，需要在两台设备上分别都执行一次以上操作，如果不行，重启chrome浏览器再重复以上操作
